# excel-data-analysis-dashboard
Sales Dashboard Using Excel Pivot Tables
📊 Project Overview

This project demonstrates the creation of an interactive Sales Dashboard in Microsoft Excel using Pivot Tables, Pivot Charts, and Slicers. The dashboard provides a comprehensive view of sales performance, customer insights, regional analysis, and transaction trends.

The objective of this project is to transform raw sales data into meaningful business insights through data visualization and reporting techniques.

🚀 Features
Sales Trend Analysis
Monthly sales performance visualization.
Tracks revenue trends throughout the year.
Identifies peak and low-performing months.
Salesperson Performance
Compares sales generated by individual sales representatives.
Highlights top-performing salespersons.
Regional Sales Analysis
Displays sales distribution across regions.
Enables regional performance evaluation.
Customer Analysis
Identifies top customers based on revenue contribution.
Helps understand customer purchasing behavior.
Transaction Analysis
Analyzes invoice transaction count by revenue ranges.
Provides insights into transaction distribution.
Interactive Filters
Region Slicer
Salesperson Slicer

Users can dynamically filter dashboard data for customized analysis.
